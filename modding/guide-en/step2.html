Step 2: Getting into the mess

This part will be either hard or easy, depending on your coding experience.

If you don't have experience, visit this. It's an (external) guide that
explains how to use LOVE2D and Lua.

Anyway, you now encounter a screen with lots of
weird buttons and a text editor on the center.

This is Notepad++, your coding friend that will help you code a lot.

And now is the moment to go

apeshit.

Do whatever you want! Change the font, the sound effects,
the textures... You can even replace text inside the game itself, like cell names!

With a lot of determination, you can even transform into the next Monomod if you want!

Step 2.1: Compiling

To actually apply your changes to the game, you need to run the batch file we saved earlier!

It will automatically transform your messy code into a usable .love format!
